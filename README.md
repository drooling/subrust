## Instructions

#### Compile Source

```
git clone https://github.com/drooling/subrust && cd subrust
cargo run --release
```

#### Use release

- Navigate to [releases](https://github.com/drooling/subrust/releases)
- Download release ( `subrust.zip` )
- Extract archive
- Open console in directory
- Run `subrust`
